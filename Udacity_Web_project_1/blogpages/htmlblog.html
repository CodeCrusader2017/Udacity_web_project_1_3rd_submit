Update 15th July 2021:

In HTML, inline elements are meant to differentiate part of a text, to give it a particular function or meaning. Inline elements usually comprise a single or few words. Inline elements are: links, emphasized words, important words. SPANs, BLOCKQUOTE, Unordered lists - use ul and li, Ordered lists - use ol and li, Typical HTML elements: Attributes, id, src, href.
